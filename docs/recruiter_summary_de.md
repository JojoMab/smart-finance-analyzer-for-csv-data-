# Recruiter Summary

## Kurzbeschreibung

Python-Projekt zur Analyse synthetischer Finanzdaten mit Budgetampel, Validierung und Report-Ausgabe fuer Finanz-/Versicherungs-IT.

## Sichtbare Kompetenzen

- Python-Grundlagen
- Datenanalyse
- Validierung
- Reporting
- Tests
- GitHub Actions
- Business-/IT-Bezug

## Datenhinweis

Die Daten sind synthetisch und dienen nur der Demonstration.
